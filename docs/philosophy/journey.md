# NEST Journey — 人はどう育つか

> 🧭 **ここは思想層です。** この文書はNESTで育つ人のための**地図（Journey）**です。

- **Version**: 1.0
- **発行**: AT-Labo

---

## この地図の約束

- **これは地図であって、関所ではない。** どのレベルの人も、NESTのすべてを見てよい。鍵のかかった扉はひとつもない
- **自己申告制。** クリアしたかどうかは、あなたが決める。試験も審査もない
- **順番どおりでなくていい。** 飛ばしても、戻っても、同じ場所に長くいてもいい。地図は道を示すだけで、歩幅はあなたのものだ
- **現在地の記録**: いまの自分のレベルを `tasks/STATUS.md` に一行書いておくと、次に来たとき迷わない（例: `Journey: Level 3`）

## 成長の地図 — Level 0 から 7 へ

### Level 0. こんにちは
- **ゴール**: PortalとSTART_HEREを眺めた
- **最初の一歩**: 何もしなくていい。ようこそ
- **使う場所**: Portal
- **詰まったら**: 詰まりようがないので安心してほしい
- **次へ**: NESTの中身が5つだけなことを知る

### Level 1. NESTを知る
- **ゴール**: 5つの場所（Portal・Handbook・Knowledge・Apps・STATUS）の名前と役割が言える
- **最初の一歩**: START_HEREを10分読む
- **使う場所**: START_HERE
- **詰まったら**: 覚えなくていい。忘れたらまた見ればいい
- **次へ**: AIに挨拶してみる

### Level 2. AIと話してみる
- **ゴール**: AIにプロンプト（AIへの声かけ文）を貼って、返事をもらえた
- **最初の一歩**: `prompts/` にあるはじめまして用のプロンプトをコピーして、AIに貼る
- **使う場所**: prompts/
- **詰まったら**: AIの返事が難しかったら「もっとやさしく説明して」と言っていい
- **次へ**: そのAIと一緒に、何か小さいものを作ってみる

### Level 3. 最初のアプリを作る
- **ゴール**: AIと一緒に作った小さなアプリが、目の前で動いた
- **最初の一歩**: 「じゃんけんゲーム作りたい」のような一言をAIに伝える（最初の1個は仕様書もタスクも不要。作る楽しさが先）
- **使う場所**: apps/＋AIパートナー
- **詰まったら**: エラーの画面をそのままAIに見せる。一緒に読めばいい
- **次へ**: さっき裏で何が起きていたのかを知る

### Level 4. Gitを知る
- **ゴール**: 「さっきAIがやっていたこと（記録・保存・巻き戻しの仕組み）」を自分の言葉で説明できる
- **最初の一歩**: Handbookの「Gitとの付き合い方」を読む。体験した後だから、答え合わせとして読める
- **使う場所**: Handbook
- **詰まったら**: NotebookLMやAIに「Gitって結局何？」と聞いていい。何度でも
- **次へ**: 作ったものを世界に置いてみる

### Level 5. 公開する
- **ゴール**: 自分のアプリのURLを持っている
- **最初の一歩**: Handbookの公開手順に沿って進める（人間にしかできない操作は一覧がある。ひとつずつで大丈夫）
- **使う場所**: Handbook＋GitHub
- **詰まったら**: Knowledgeを引く。先輩も同じ場所で転んでいる
- **次へ**: そのURLを、誰かに送ってみる

### Level 6. 人に共有する
- **ゴール**: 誰かに見せて、感想をもらった
- **最初の一歩**: 一番怖くない相手にURLを送る。「作ってみたんだ」の一言と、質問をひとつ添えると感想が返ってきやすい（例:「どこが一番使いにくい？」）
- **使う場所**: あなたの人間関係（NESTの外！）
- **詰まったら**: 反応が薄くても大丈夫。動くものを作って見せた時点で、もう十分すごい
- **次へ**: 受け取ってきたものを、次の誰かへ返す番

### Level 7. 巣立ち — 自分の巣をつくる
- **ゴール**: 自分の巣（自分のNEST）を持ち、次の誰かを迎えられる
- **最初の一歩**: Starter Kitで自分の巣を作る。育ったアプリを独立させるのもいい
- **使う場所**: Starter Kit
- **詰まったら**: いつでもこの巣に戻ってきていい。巣立ちは絶縁ではない
- **次へ**: あなたの巣に、新しいLevel 0の誰かがやってくる。今度は、あなたが迎える番

## 地図は、円環である

この地図は、Level 7で終わる一本道ではない。

```
        Level 0 こんにちは ←──────────┐
           ↓                          │
        Level 1〜6 育つ               │ 迎える
           ↓                          │
        Level 7 巣立ち ────────────→──┘
```

巣立った人が新しい巣を作り、そこに新しいLevel 0の誰かが訪れる。かつて迎えられたあなたが、今度は迎える側になる——「**共に育てる**」はこの循環のことであり、受け取った**恩恵**を返し、迎えてくれた人への**感謝**を次の人への優しさに変える仕組みが、この地図そのものである。

だからLevel 7は卒業ではない。**この地図でいちばん豊かな場所**だ。

## この地図の育て方

8つのレベルの**骨格（何ができるようになるか）は変えない**。各レベルの**中身（使う道具・手順）は時代に合わせて入れ替えてよい**。骨格の変更は憲法改正と同じ扱いとする（ADR必須）。
