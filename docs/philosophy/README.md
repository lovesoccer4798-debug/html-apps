# NEST 思想層（Philosophy）

> 🧭 **ここはNESTの思想層です。** NESTの「変わらない部分」— 判断のよりどころとなる6つの文書がここにあります。

## 6つの文書と役割

```
                 ┌────────────────────────────┐
   なぜ生まれた？ │  💗 Story（心）              │
                 └─────────────┬──────────────┘
                               ▼
                 ┌────────────────────────────┐
   何を信じる？   │  🔥 Manifesto（旗）          │
                 └─────────────┬──────────────┘
                               ▼
                 ┌────────────────────────────┐
   どこへ向かう？ │  🔭 Vision（望遠鏡）         │
                 └─────────────┬──────────────┘
         ┌─────────────────────┴─────────────────────┐
         ▼                                           ▼
┌─────────────────────┐                 ┌─────────────────────┐
│ ⚖️ Brand Book（憲法）│    対になる     │ 📐 Design Principles │
│ 人として、どうあるか  │ ←──────────→   │ （定規）             │
│                     │                 │ 物として、どうあるか  │
└──────────┬──────────┘                 └──────────┬──────────┘
           └─────────────────┬─────────────────────┘
                             ▼
                 ┌────────────────────────────┐
   人はどう育つ？ │  🗺 Journey（地図）          │
                 └────────────────────────────┘
```

| 文書 | 例えるなら | 答える問い | 変わる頻度 |
|---|---|---|---|
| [Story](story.md) | 心 | なぜ生まれた？ | ほぼ不変 |
| [Manifesto](manifesto.md) | 旗 | 何を信じる？ | ほぼ不変 |
| [Vision](vision.md) | 望遠鏡 | どこへ向かう？ | ほぼ不変 |
| [Brand Book](brand-book.md) | 憲法 | 人としてどうある？ | 憲法改正のみ |
| [Design Principles](design-principles.md) | 定規 | 物としてどうある？ | 年輪のように成長 |
| [Journey](journey.md) | 地図 | 人はどう育つ？ | 骨格は不変 |

**思想層はこの6つで完結する。** 7つ目の文書は作らない（問いが出尽くしているため）。新しい思想が必要になったら、まず既存6文書への追記で表現できないかを検討する。

## 読む順番

はじめての方は上から順に: **心 → 旗 → 望遠鏡 → 憲法 → 定規 → 地図**。
全部読まなくてもかまいません。必要になったとき、必要な一冊だけで大丈夫です。

## 迷ったときの引き方

| 迷い | 引く文書 |
|---|---|
| 文章の言い回し・態度に迷った | ⚖️ Brand Book（憲法） |
| 機能を足すか・削るかに迷った | 📐 Design Principles（定規） |
| やる気を失った・原点に戻りたい | 🔥 Manifesto（旗）と 💗 Story（心） |
| この先どうしたいんだっけ、と思った | 🔭 Vision（望遠鏡） |
| 自分（学ぶ人）の現在地を見失った | 🗺 Journey（地図） |

## 改訂ルール

- Philosophy・価値観・六箇条・原則・Journeyの骨格の変更は「**憲法改正**」。必ずADR（設計判断の記録）を残してから行う
- 言い回し・実例・説明は、より伝わる言葉が見つかり次第、遠慮なく育てる
- この思想層自身が「育てる」の見本であること
