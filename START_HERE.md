# 🚀 START_HERE — ようこそ、NESTへ

> ここは、はじめての方のための10分の案内です。読み終わるころには、NESTの全体像がわかり、AIとの最初の一歩を踏み出せます。

## まず、この画面について（30秒）

いまあなたは **GitHub（ギットハブ）** というサイトにいます。世界中の人が作ったものを保管する、巨大な倉庫のような場所です。英語のボタンがたくさん見えて不安かもしれませんが、**大丈夫。触らなくていいです。** 読むのは、この真ん中の文章だけ。青い文字のリンクをクリックして移動するだけで、すべて回れます。壊れることは何もありません。

## NESTとは（30秒）

NESTは、**プログラミング未経験でも、AIと一緒にアプリを作れるようになる場所**です。合言葉は——

> **「安心して創造し、共に育てる。」**

NEST＝巣。ここは住みつく場所ではなく、孵って、育って、いつか飛び立つための場所です。（生まれた理由は [NEST Story](docs/philosophy/story.md) に書いてあります。3分で読めます）

## 1. NESTの中身は、5つだけ（2分）

覚えなくて大丈夫。忘れたらまたここを見ればいいので。

| 場所 | 一言でいうと |
|---|---|
| 🏠 **Portal** | NESTのホーム画面。どこに何があるかが見える |
| 📖 **Handbook** | 使い方とルールの説明書 |
| 💡 **Knowledge** | みんなの気づきメモ。転んだ記録と立ち上がり方 |
| 🌱 **Apps** | 育てているアプリたち |
| 📍 **STATUS** | いま、なにしてる？の現在地メモ |

そして場所とは別に、いつも隣にいる同伴者が2人 — 🤖 **AIパートナー**（一緒に作ってくれる相棒）と 📓 **NotebookLM**（文章で質問できる先生）です。

## 2. たった3つの、やくそく（2分）

1. **わからないことは、恥ではありません。** いつでも、何度でも、AIに質問していいです
2. **やったことは、ファイルに残します。** 会話は消えますが、ファイルは残ります。残せば、明日の自分が助かります
3. **迷ったら、📍STATUS を見ます。** 「いま何をしていたか」がいつも書いてあります

## 3. AIパートナーを迎える（2分）

NESTでは、AIが隣に座ってくれます。**どのAIでも構いません**（NESTは特定のAIに依存しない作りになっています）。まだAIを使ったことがなければ、無料で始められるチャットAIをひとつ用意してください — いま使える具体的な選択肢は [AIツール案内](docs/ai-tools.md) にまとめてあります。

## 4. 最初の10分を、やってみる（3分）

手順は3つだけです。

1. [`prompts/hello-nest.md`](prompts/hello-nest.md) を開き、中のプロンプト（AIへの声かけ文）を**まるごとコピー**する
2. あなたのAIに**貼り付けて送信**する
3. AIが自己紹介と質問を返してきたら——おめでとうございます、それが最初の一歩です。そのまま会話を続けて、小さなものを一緒に作ってみましょう（次は [`prompts/first-app.md`](prompts/first-app.md) の出番です）

## 6. こわれても、だいじょうぶ（30秒）

NESTのすべての変更は **Git（ギット）** という仕組みが自動で記録しています。どんな失敗をしても、いつでも前の状態に戻せます。**あなたがここで壊せるものは、何もありません。** だから、安心して触ってください。
