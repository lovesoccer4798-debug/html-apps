# 仕様: NEST Dashboard v1（毎日使う人のホーム）

- **作成日**: 2026-07-07
- **最終更新**: 2026-07-07
- **ステータス**: 実装済み
- **関連**: `portal-v1.md`, `docs/vocabulary.md`（4つの役割）, `docs/roadmap.md`（Phase 5）

## 1. 目的（なぜ作るのか）

2回目以降の利用者（毎日使う人）が、既存のすべての情報へ2クリック以内で到達できるホームを作る。**新しい機能は作らない。情報の整理だけを行う。**

## 2. スコープ

### やること

- Portalの2ページ目 `dashboard.html` として実装（6つ目の「場所」にはしない — START_HEREの「5つだけ」の約束を守る）
- 6グループの整理: いまを知る／つくる／まなぶ／しらべる／AIと／よりどころ
- Portalヘッダーに「🧭 2回目以降の方はこちら」ボタンを追加
- CSSを `style.css` に分離（2ページで共有。規模が育ったら分割する、の規約どおり）

### やらないこと

- JavaScript・検索・動的な情報表示・新規コンテンツの執筆
- **Workspace Versionの数字の手書き表示**（リリースのたび古くなる嘘の温床。CHANGELOG/Releasesへのリンクで代替 — 鉄則2「手書きの複製ゼロ」）

## 3. ユーザーから見た振る舞い

Portal→「2回目以降の方はこちら」→Dashboard。6枚のカードに整理されたリンク一覧。すべて既存ページへのリンクで、押したら必ず目的地に着く。

## 4. エッジケース

- JS無効: 全機能動作（JSゼロ）／スマホ幅: カード1列化／ダーク: 自動追従（style.css共有）

- [x] 新機能ゼロ・JSゼロ・手書きの複製ゼロ（Version数字を書いていない）
