# Task Calendar

毎日のタスクを「日・週・月」の3つの視点で眺めながら、チェックボックスでこなしていく個人用タスクカレンダー。所要時間を設定したタスクはリング型カウントダウンタイマー（フォーカス画面）で集中して取り組める。完了の積み上げ（完了率・連続日数・週グラフ）を「振り返り」で見えるようにして、習慣づけを手伝う。

- 繰り返しタスク（毎日・毎週・毎月・毎年）と、タスクとは別の「予定」を追加できる
- 行を左へスワイプすると「編集／削除」
- 設定画面でライト／ダークのテーマと、選択枠などのアクセントカラーを選べる
- デザインはオーナー作成のhandoff準拠（`specs/task-calendar-v2.md`・ADR参照）

## 使い方・起動方法

`index.html` をブラウザで開くだけ（ビルド不要・依存ゼロ・データはブラウザの `localStorage` に保存）。

GitHub Pages で公開している場合は **PWA** として使える：スマホでURLを開き「ホーム画面に追加」（iOS Safariは共有ボタン→ホーム画面に追加、Androidはメニュー→アプリをインストール）。以後はアプリのように起動でき、オフラインでも開ける。

1. **日**ビューでタスクを追加する（所要時間は任意。5/15/25/45/60分のプリセットあり）
2. 終わったらチェックボックスで完了。所要時間つきのタスクは「開始」でカウントダウンタイマーが起動する
3. **週**・**月**ビューで進み具合を振り返る（月ビューは完了がドットで積み上がって見える）

## テスト・動作確認の方法

自動テストはなし（`docs/testing-policy.md` の方針どおり手動確認）。確認する主要フロー:

- タスクの追加（空文字は拒否）・完了／解除・削除（「元に戻す」で復元）
- リロード後もデータが残る
- 日／週／月の切替、◀▶と「今日」での移動
- タイマーの開始・一時停止・リセット・時間切れ表示・「完了にする」
- ライト／ダークテーマ両方で表示が破綻しない
