# 仕様: Task Calendar v3 — 長く使うための振り返り強化（ピッカー・年ビュー・期間振り返り・メモ・フォント）

- **作成日**: 2026-07-15
- **最終更新**: 2026-07-15
- **ステータス**: 実装済み
- **関連**: v2仕様 `task-calendar-v2.md` / タスク `tasks/in-progress/20260714-task-calendar.md`

## 1. 目的

「今日をこなす」道具から「何年も使って振り返れる」道具へ。過去へ自由に飛べる導線（年月ピッカー）、1年の積み上げが一目で見える年ビュー（ヒートマップ）、週／月／年の振り返り、日記になるタスクメモ、好みのフォントを加える。**v2の見た目・操作は壊さない（すべて追加のみ）。**

## 2. スコープ

### やること

1. **年月ピッカー**: 画面上部の「2026年7月」タイトルをタップ→ボトムシート。年を‹›で送り、1〜12月のグリッドから選んで日／週／月／年ビューのままジャンプ。「今日へ戻る」ボタンつき
2. **年ビュー**: セグメントに「年」を追加。1月〜12月を行にした**ヒートマップ**（1日=1マス、完了数で4段階の濃さ）。年の完了数・実行日数の統計行つき。マスをタップでその日のデイリーへ
3. **振り返りの期間切替**: 振り返り画面の上に「週｜月｜年」セグメント。完了率・完了数・グラフ（週=曜日別／月=週別／年=月別）・できたことリスト（直近15件・日付つき）が期間に追従。連続達成と「今日のひとこと」は共通
4. **タスクメモ**: 追加／編集シートに「メモ（任意）」。カードのタイトル下に1行表示、振り返りの「できたこと」にも表示（日記の代わりになる）。繰り返しタスクは**編集時にその日の分のメモ**として保存（`memoDates[日付]`。追加時のメモは全日共通の基本メモ）
5. **フォント選択**: 設定に「ゴシック／丸ゴシック／明朝／等幅」の4択。端末内蔵フォントのみ使用（CDN・Webフォント同梱はしない。機種により見え方が変わることを設定画面に明記）

### やらないこと

- ルーティンパッケージ（v4）・ビジョンボード（v5）は別仕様
- 過去期間へのナビゲーション付き振り返り（v3は「今週・今月・今年」のみ。過去日はピッカーで到達できる）

## 3. データ・入出力

- 保存先は `nest.task-calendar.v2` のまま。**追加フィールドのみ**（後方互換。v2に戻してもデータは壊れない）
  - `settings.font`: `"gothic" | "rounded" | "mincho" | "mono"`（既定 gothic）
  - タスク・予定に `memo`（文字列・任意）、繰り返しタスクに `memoDates`（日付→文字列）
