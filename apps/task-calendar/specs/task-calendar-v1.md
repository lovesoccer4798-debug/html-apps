# 仕様: Task Calendar v1 — タスクカレンダー

- **作成日**: 2026-07-14
- **最終更新**: 2026-07-14
- **ステータス**: 実装済み
- **関連**: タスク `tasks/in-progress/20260714-task-calendar.md`

## 1. 目的（なぜ作るのか）

毎日のタスクを「今日・今週・今月」の3つの視点で眺めながら、チェックボックスでこなしていく個人用タスクカレンダー。完了したタスクが積み上がっていくのを振り返れるようにすることで自己肯定感を育て、習慣づけを手伝う。タスクに設定した時間をカウントダウンするタイマーで「はじめる」ハードルも下げる。

## 2. スコープ

### やること

- **3つのビュー**: デイリー（1日のタスク一覧）／ウィークリー（7日分を横断）／マンスリー（月間カレンダー）。ワンタップで切替できる
- **タスクの追加・完了・削除**: デイリービューで追加。チェックボックスで完了／解除。削除は「元に戻す」つきで取り返しがつく（設計原則3）
- **所要時間の設定とタイマー**: タスクに任意で所要時間（分）を設定できる。「開始」を押すと全画面のカウントダウンタイマー（円形プログレス・一時停止／リセット・終了時にそのまま完了にできる）
- **振り返り（自己肯定感の装置）**:
  - ヘッダーに「今日の完了数」「連続日数（1日1つ以上完了した日の連なり）」「累計完了数」を常時表示
  - 月間ビューは完了タスクをドットで可視化し、全部完了した日はセルが色づく
  - その日のタスクを全部終えると、ねぎらいのバナーが出る
- **保存**: `localStorage` に保存し、リロード後も保持する
- **PWA対応**: スマホのホーム画面に追加でき、一度開けばオフラインでも動く（manifest＋Service Worker。ホスティングはGitHub Pages想定＝無料）
- **デザイン**: テーマカラーはオーナー指定のエメラルドグリーン `#00C896`。アイコンは Lucide のインラインSVG（絵文字を使わない）。依存ライブラリ・CDNなし

### やらないこと（明示的に対象外とするもの）

- タスクの編集（v1では削除→再追加で代替）
- 繰り返しタスク・リマインダー通知・時刻（何時開始）の指定
- 複数デバイス間の同期・アカウント機能
- ドラッグ＆ドロップの並べ替え
- Google カレンダー等の外部サービス連携

## 3. ユーザーから見た振る舞い

```
┌──────────────────────────────────────────────┐
│ ← NEST Portal へ戻る                          │
│ [icon] Task Calendar                          │
│ 今日 2/5 完了 ・ 連続 3日 ・ 累計 42          │  ← エメラルドのヒーロー帯
├──────────────────────────────────────────────┤
│ [日｜週｜月]        ◀  今日  2026年7月14日（月） ▶ │
├──────────────────────────────────────────────┤
│ ＋ タスクを追加 [＿＿＿＿＿] [25]分 (5/15/25/45/60)│
│ ▓▓▓▓▓░░░░░ 2/5                                │
│ ◉ 朝のストレッチ            15分  ▶開始  🗑    │
│ ○ 企画書のたたき台          45分  ▶開始  🗑    │
└──────────────────────────────────────────────┘
```

- **日**: その日のタスク一覧＋追加フォーム＋進捗バー。チェックで完了（ポップするアニメーション）。全部完了でねぎらいバナー
- **週**: 日〜土の7列。各列でもチェック操作ができる。日付をタップするとその日のデイリービューへ
- **月**: カレンダーグリッド。タスクはドット（完了＝塗り、未完了＝枠）で表示。日付タップでデイリービューへ
- **タイマー**: 所要時間つきタスクの「開始」を押すと全画面オーバーレイ。残り時間（mm:ss）と円形プログレスが減っていく。一時停止／再開／リセット／途中でも「完了にする」が可能。0になると静かなチャイムと「おつかれさま！」表示
- **日付ナビ**: ◀▶で前後（日／週／月単位）、「今日」で現在に戻る

## 4. データ・入出力

- 保存先: `localStorage` キー `nest.task-calendar.v1`
- 構造:

```json
{
  "tasks": [
    {
      "id": "t1720900000000abc",
      "title": "朝のストレッチ",
      "date": "2026-07-14",
      "minutes": 15,
      "done": false,
      "doneAt": null,
      "createdAt": 1720900000000
    }
  ]
}
```

- 制約: `title` は必須（空白のみは拒否）・120文字まで。`minutes` は任意・1〜600の整数。`date` はローカルタイムゾーンの `YYYY-MM-DD`

## 5. エッジケース・エラー時の挙動

- 空タイトルで追加: 追加せず、入力欄を振動（shake）させてフォーカスを戻す
- `localStorage` が使えない／壊れたJSON: 空の状態から開始し、以後はメモリ上でのみ動作（アプリは落とさない）
- 所要時間のないタスク: 「開始」ボタンを表示しない
- タイマー起動中にもう一度別タスクを開始: 先に開いているタイマーを閉じてから（オーバーレイ表示中は背後を操作できない）
- タブが非アクティブでもタイマーは実時間基準（終了時刻方式）で正しく進む
- 削除の取り消し: 削除直後6秒間「元に戻す」トーストを表示

## 6. 完成の定義（Definition of Done）

- [x] 日／週／月の3ビューが切り替えられ、◀▶と「今日」で日付移動できる
- [x] タスクを追加・完了・完了解除・削除でき、リロード後も保持される
- [x] 空文字での追加は拒否される
- [x] 削除は「元に戻す」で取り消せる
- [x] 所要時間つきタスクは「開始」からカウントダウンタイマーが起動し、一時停止／リセット／完了ができ、0で終了表示になる
- [x] 今日の完了数・連続日数・累計完了数が表示され、操作に追従して更新される
- [x] 月間ビューに完了状況がドットで表示される
- [x] PWAとしてインストール可能（manifest・アイコン・Service Worker）で、オフラインで再訪してもアプリが開く
- [x] ライト／ダークの両テーマで破綻しない
- [x] テスト方針（`docs/testing-policy.md`）に沿った手動確認（ブラウザで主要フローを一巡）が完了している
