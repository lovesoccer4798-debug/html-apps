# Changelog

このアプリのユーザーに見える変更を記録する（[Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) 形式・[Semantic Versioning](https://semver.org/lang/ja/) 準拠）。

## [1.24.0] - 2026-07-18

### Added

- ルーティンを「予定」型でも作れるように（これまではタスク＝習慣のみ）。例：「仕事(Realation)」で 出勤＝月〜金・休み＝土日 を登録すると、休みが分かりやすい。予定型は達成トラッキングなし・「月」表示ON/OFFも同様に効く
- ルーティンの各項目に「曜日で」を追加。月〜日から入れたい曜日を選べます（例：出勤＝月〜金、ジム＝火・木）
- 「月」ビューに、ルーティンをその場で表示/非表示できるチップを追加（設定に潜らず1タップ）。オフでも一覧・日/週などには残ります

## [1.23.0] - 2026-07-18

### Added

- 予定・タスクに「『月』カレンダーにも表示する」ON/OFFを追加（追加・編集シート）。オフにすると「月」カレンダーの見た目からは消えますが、その日の予定・タスク一覧や 日/週/時間/年 には残ります。カレンダーの見栄えを整えたいときに
- ルーティンにも同じ「『月』カレンダーにも表示する」ON/OFFを追加（ルーティン編集）。毎日の習慣で「月」が窮屈になるとき、月カレンダーからは隠しつつ一覧・他ビューには表示できます。※フィルタチップの「ルーティン」を外すと従来どおり全ビューから隠れる挙動もそのまま使えます

## [1.22.0] - 2026-07-18

### Added

- 睡眠記録に「お昼寝」を追加。時刻ではなく「何分」で入力でき、その日の合計睡眠（夜＋昼寝）に反映されます（振り返りの平均・記録一覧にも合算）
- 月ビューで、対象月ではない日（前後の月）の予定・タスクを薄く表示。今の月がはっきり目立つように

### Changed

- タイマー完了時の時刻の自動反映を、実際に動かした時間に忠実に。終了＝完了した実時刻、開始＝タイマーを始めた実時刻（どちらも最寄りの5分）、タイマー時間＝その差。すでに時刻が入っていても上書きし、設定より長く動かした超過ぶんも反映されます（例：11:00に30分タイマー→11:57完了なら 開始11:00・終了11:55・55分）

## [1.21.1] - 2026-07-18

### Changed

- 月ビュー（予定表）で、日をまたぐ予定を隙間なく連日つながる帯に（TimeTree風）。週の端は角丸で区切り、途中は完全につながって見えます
- 月ビューの日と日のあいだの余白を詰めて、すっきりした見た目に

### Fixed

- パッケージ作成カードの日付入力が中央寄せで右側が空いて見えていたのを、左寄せに統一（iOSの日付表示の見た目を修正）

## [1.21.0] - 2026-07-18

### Added

- 日をまたぐ予定に対応（旅行・お盆帰省など）。予定の追加・編集で「終了日」を入れると、開始日から終了日まで日にちを横断してつながって表示されます（TimeTree／Googleカレンダー風）。月ビューでは色つきの帯が連日つながり、週の始まりで名前をくり返し表示。日ビューでは「◯日目／全◯日」と期間（例：7/19〜7/23）を表示

## [1.19.0] - 2026-07-18

### Added

- パッケージ機能（ルーティン画面内の新タブ）。お気に入りの1日のタイムスケジュールを名前をつけて保存し、別の日にまるごとコピーできます。「取り込む日付」を選ぶとその日の予定・タスクが1つのパッケージにまとまり、「この日に入れる」で好きな日付へ複製。いらない項目は個別に外せます
- タスク一覧ページ（ボトムナビに「タスク」を追加）。日・週・時間・月・年とは別に、タスクだけを日付・時刻つきで一覧表示。未完了/すべての切替、単発タスクはその場でチェック、タップでその日へジャンプ

## [1.18.0] - 2026-07-18

### Added

- ヘッダーの検索アイコン横に「テーマ切替」ボタン。どの画面からでも 自動→ライト→ダーク をワンタップで切替（設定に行かなくてもOK。設定のテーマ選択とも同期）
- カレンダー上部（日付・日/週/時間/月/年タブ）を固定表示に。下のリストだけスクロールするので、いつでもビュー切替や日付移動ができる
- 「日」ビューの予定と予定の“あいだ”に、空き時間を薄く表示（例：「2時間00分のあいだ」）。縦の時間軸の線もつなげて、時間が流れている感じに
- 「日」ビューの最下部に、その日まるごとの感想を書く「あのね。ノート」。書いた内容はNotion連携の日記にも一緒に送られます
- 「月」ビューの予定に「くっきりフチ」をつける設定。さらに名前や意味（例：「村山さん」）に色を割り当てると、その言葉を含む予定にその色のフチがつく。共有カレンダーでも色分けは自分の画面だけで、相手には影響しません

### Fixed

- Notion中継（Cloudflare Worker）が、トークン・合言葉の前後に紛れ込んだ空白や改行を自動で除去するように（貼り付けミスによる認証エラーを起きにくく）

## [1.16.0] - 2026-07-18

### Added

- Notion連携: 日々の記録（日記・ひとことメモ・できたこと数・就寝/起床）を自分のNotionデータベースへ1日1ページで自動転記。設定 → Notion連携 で Worker URL・合言葉・データベースID を入れてON。手動送信ボタンもあり。同じ日付は上書き（重複しない）

## [1.15.0] - 2026-07-18

### Added

  - 空いている時間帯をタップ→その日時が入った追加シート、ブロックをタップ→編集。今日の列には現在時刻の赤い線
- 検索（カレンダー画面の虫めがね）: タスク・予定・メモ・日記をキーワード検索。候補の右に年月日と曜日が出て、タップでその日のデイリーへジャンプ（繰り返しは直近の予定日へ）

### Fixed

- 土曜=青・日曜/祝日=赤の色がどのビューにも反映されていなかった問題（色の変数定義が抜けていた）
