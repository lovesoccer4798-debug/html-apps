# Changelog

このアプリのユーザーに見える変更を記録する（[Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) 形式・[Semantic Versioning](https://semver.org/lang/ja/) 準拠）。

## [0.9.0] - 2026-07-16

### Added

- Googleログインとクラウド同期（Firebase Phase A）: 設定の「アカウントと同期」からログインすると、タスク・予定・ルーティン・目標・睡眠・メモ等が自分専用のクラウド領域へ自動バックアップされ、別端末でも復元できる（ビジョンボードの写真は端末内のみ）
- オフライン時はローカル動作を継続し、復帰時に同期。ログアウトしても端末内データは残る
- Firebase SDK（v10.14.1 compat）を同梱・遅延読み込み（CDN不使用・ADR: 20260716-firebase-sdk-vendoring）

## [0.8.0] - 2026-07-15

### Added

- ビジョンボード（ルーティンタブ内「ビジョン」切替）: テンプレート型（4レイアウトのコラージュ枠）と自由配置型（写真・ことばをドラッグで自由に貼る、拡大縮小・回転つき）
- 写真ごとのキャプション（意図・テーマ）とボード全体のメモ
- 写真はIndexedDBに自動縮小（長辺1280px）で保存 — localStorageを圧迫せずオフラインOK。上限30枚
- 振り返りに「1年前（半年前・1ヶ月前）の今日のひとこと」カード

## [0.7.0] - 2026-07-15

### Added

- マイカレンダー: 自由に追加できる分類（最大8個・名前と色を設定）。タスク・予定の追加時に所属を選択
- TimeTree風フィルタチップ（すべて／各カレンダー／ルーティン）— チェックしたものだけ表示、選択は保存
- 設定に「マイカレンダー」管理（追加・名前編集・色切替・削除=項目はマイカレンダーへ移動）
- 項目の色タグ・予定表ラベルにカレンダー色を反映（個別色・ルーティン色が優先）

## [0.7.1] - 2026-07-15

### Changed

- 日付見出し（日/週/月/年の左上）のサイズを20pxに統一し、見た目に統一感を持たせた

### Fixed

- Service Workerファイルが空になっていた不具合を修正（オフライン起動の回復）

## [0.6.0] - 2026-07-15

### Added

- 睡眠記録: 日ビューに就寝・起床の入力（日ごとに保存）。振り返りに平均就寝・平均起床・平均睡眠時間（週/月/年で集計）
- 文字サイズ設定（大=今のまま・中・小）
- 土曜=青・日曜=赤の色分け（月・週の曜日と日付数字）＋日本の祝日を赤表示（固定祝日・ハッピーマンデー・春分秋分・振替休日）

### Fixed

- 予定表モードでセルの高さ・幅が内容によってズレる問題（全セル均等・はみ出しはクリップ）

## [0.5.1] - 2026-07-15

### Added

- 日・週・月・年ごとの「目標」— カレンダー見出し下の行をタップして、その期間の目標を気分で書ける（期間ごとに保存）

### Fixed

- 更新直後に新旧ファイルが混ざって一部ボタン（ルーティン等）が効かなくなる問題 — 資材にバージョン番号を付けて配信するよう変更

## [0.5.0] - 2026-07-15

### Added

- 月カレンダーに「予定表」モード（TimeTree風）: ドット表示はそのままに、切替で日付の下に色つきラベル最大4件を表示
- タスク・予定に色を設定可能（追加／編集シート。予定表モードのラベルとカードの色タグに反映）
- 追加／編集・ルーティン・共有シートに「×」閉じるボタン（何も追加せずに戻れる）

### Fixed

- 週見出し（例: 7月27日〜8月2日）が中途半端な位置で折り返す問題（週表示のみ文字サイズを調整＋改行禁止）

## [0.4.0] - 2026-07-15

### Added

- ルーティンパッケージ（ボトムナビ4つ目のタブ）: ゴール文・色・週の目標日数（既定3日）つきの習慣の束
- ゆる達成: パッケージ内のどれか1つでも完了したら「やった日」。週の目標日数で「今週合格」＋連続クリア週数
- 有効化／一時停止（実績は残る）、直近12週の合否履歴、カレンダー上のパッケージ色タグ
- 共有: JSONテキストのコピー＆貼り付けでルーティンを交換（読み込みは一時停止状態で追加）
- 振り返り画面にルーティン別の今週達成カード

## [0.3.1] - 2026-07-15

### Fixed

- フォント選択が端末で反映されたか分からない問題: フォールバック強化・設定画面にプレビュー文・端末に日本語書体がない選択肢へ「＊」表示（iPhoneは明朝・等幅の日本語書体がなく数字・英字のみ変わる）

## [0.3.0] - 2026-07-15

### Added

- 年月ピッカー（タイトルタップで過去・未来の任意の年月へジャンプ、「今日へ戻る」つき）
- 年ビュー（1日=1マスのヒートマップ。完了の濃淡・年の完了数・実行日数。マスからデイリーへ）
- 振り返りの期間切替（週／月／年 — 完了率・完了数・グラフ・できたことが追従）
- タスク・予定のメモ（カードと振り返りに表示。繰り返しタスクは日ごとのメモ＝日記になる）
- 設定にフォント選択（ゴシック／丸ゴシック／明朝／等幅。端末内蔵フォントのみ）

## [0.2.0] - 2026-07-15

### Added

- 設定画面（テーマ: 自動／ライト／ダークの手動切替、アクセントカラー6色 — カレンダーの選択枠・チェック・ボタンに反映）
- 繰り返しタスク（毎日・毎週・毎月・毎年）。日ごとに独立して完了でき、削除は「この回のみ／すべて」を選べる
- タスクとは別の「予定」（チェック・タイマーなしのその日の予定）
- 行の左スワイプで「編集／削除」、編集用ボトムシート
- フォーカス画面（全画面リングタイマー・つぎのタスク表示）と実行中カード。タイマーはリロード後も復元
- 振り返り画面（今日の完了率・連続達成・今週平均・週の棒グラフ・できたこと・今日のひとことメモ）

### Changed

- デザインをオーナー作成のhandoff（ブルーグレー×グリーン、モバイル基準、ボトムナビ＋タイムライン）に全面刷新（ADR: 20260715-task-calendar-v2-design）
- 週・月カレンダーは月曜はじまりに変更（handoff準拠）
- データ形式を `nest.task-calendar.v2` に移行（v1データは初回に自動移行）

## [0.1.0] - 2026-07-14

### Added

- 初版リリース
- 日／週／月の3ビューと日付ナビゲーション（◀▶・「今日」）
- タスクの追加・チェックボックスでの完了／解除・削除（「元に戻す」つき）
- 所要時間（分）の設定と、円形プログレスつきカウントダウンタイマー（一時停止・リセット・終了チャイム・そのまま完了）
- 振り返りサマリー（今日の完了数・連続日数・累計完了数）と、月ビューの完了ドット・全部できた日の色づけ
- `localStorage` への自動保存、ライト／ダークテーマ対応
- PWA対応（manifest・アプリアイコン・Service Worker）— ホーム画面に追加でき、オフラインでも開ける
