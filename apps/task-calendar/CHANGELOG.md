# Changelog

このアプリのユーザーに見える変更を記録する（[Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) 形式・[Semantic Versioning](https://semver.org/lang/ja/) 準拠）。

## [0.4.0] - 2026-07-15

### Added

- ルーティンパッケージ（ボトムナビ4つ目のタブ）: ゴール文・色・週の目標日数（既定3日）つきの習慣の束
- ゆる達成: パッケージ内のどれか1つでも完了したら「やった日」。週の目標日数で「今週合格」＋連続クリア週数
- 有効化／一時停止（実績は残る）、直近12週の合否履歴、カレンダー上のパッケージ色タグ
- 共有: JSONテキストのコピー＆貼り付けでルーティンを交換（読み込みは一時停止状態で追加）
- 振り返り画面にルーティン別の今週達成カード

## [0.3.1] - 2026-07-15

### Fixed

- フォント選択が端末で反映されたか分からない問題: フォールバック強化・設定画面にプレビュー文・端末に日本語書体がない選択肢へ「＊」表示（iPhoneは明朝・等幅の日本語書体がなく数字・英字のみ変わる）

## [0.3.0] - 2026-07-15

### Added

- 年月ピッカー（タイトルタップで過去・未来の任意の年月へジャンプ、「今日へ戻る」つき）
- 年ビュー（1日=1マスのヒートマップ。完了の濃淡・年の完了数・実行日数。マスからデイリーへ）
- 振り返りの期間切替（週／月／年 — 完了率・完了数・グラフ・できたことが追従）
- タスク・予定のメモ（カードと振り返りに表示。繰り返しタスクは日ごとのメモ＝日記になる）
- 設定にフォント選択（ゴシック／丸ゴシック／明朝／等幅。端末内蔵フォントのみ）

## [0.2.0] - 2026-07-15

### Added

- 設定画面（テーマ: 自動／ライト／ダークの手動切替、アクセントカラー6色 — カレンダーの選択枠・チェック・ボタンに反映）
- 繰り返しタスク（毎日・毎週・毎月・毎年）。日ごとに独立して完了でき、削除は「この回のみ／すべて」を選べる
- タスクとは別の「予定」（チェック・タイマーなしのその日の予定）
- 行の左スワイプで「編集／削除」、編集用ボトムシート
- フォーカス画面（全画面リングタイマー・つぎのタスク表示）と実行中カード。タイマーはリロード後も復元
- 振り返り画面（今日の完了率・連続達成・今週平均・週の棒グラフ・できたこと・今日のひとことメモ）

### Changed

- デザインをオーナー作成のhandoff（ブルーグレー×グリーン、モバイル基準、ボトムナビ＋タイムライン）に全面刷新（ADR: 20260715-task-calendar-v2-design）
- 週・月カレンダーは月曜はじまりに変更（handoff準拠）
- データ形式を `nest.task-calendar.v2` に移行（v1データは初回に自動移行）

## [0.1.0] - 2026-07-14

### Added

- 初版リリース
- 日／週／月の3ビューと日付ナビゲーション（◀▶・「今日」）
- タスクの追加・チェックボックスでの完了／解除・削除（「元に戻す」つき）
- 所要時間（分）の設定と、円形プログレスつきカウントダウンタイマー（一時停止・リセット・終了チャイム・そのまま完了）
- 振り返りサマリー（今日の完了数・連続日数・累計完了数）と、月ビューの完了ドット・全部できた日の色づけ
- `localStorage` への自動保存、ライト／ダークテーマ対応
- PWA対応（manifest・アプリアイコン・Service Worker）— ホーム画面に追加でき、オフラインでも開ける
