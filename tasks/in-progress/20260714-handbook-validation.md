## 実証終了条件（正式採用の基準 — 開始前に確定）

**以下の6つをすべて満たしたら、オーナーが正式採用を判定する。**

1. [ ] **実運用7日以上** — Handbookを毎日のホームとして使う（起点は上記10の宣言日）
2. [ ] **Creator Studioを5回以上、実戦で使う** — 生成したプロンプトを実際にAIへ貼るところまでで1回
3. [ ] **NotebookLM質問検収5問に正答** — 実証期間の終わりにもう一度5問を投げ、全問正答（1問でも誤答なら再同期→再検収）
4. [ ] **初見ユーザーテスト合格** — 知り合い1名に渡して10分観察: 成功体験に1回触れる／30秒で現在地が言える／Portal・Handbook・Dashboardの違いを自分の言葉で説明できる
5. [ ] **must（直すべき）指摘がゼロ** — 実証中に見つかった問題は下の実証ログに記録し、mustはすべて解消してから判定する
6. [ ] **チャット非依存で運用できた** — 7日間、設計時のAIチャットを一度も参照しなかった。見たくなった瞬間があれば「何の情報が欲しかったか」を実証ログに記録する（それはNESTに足りないもの＝must級の改善候補）
