# prompts/ — AI用定型プロンプト

どのAIツール（Claude Code / Codex / Gemini CLI / Cursor / Windsurf など）にもそのまま貼り付けて使える、定型プロンプトの置き場。

## 使い方

1. 目的に合うプロンプトを開く
2. `{{ }}` のプレースホルダを埋める
3. AIにそのまま貼り付ける

## 一覧

| ファイル | 用途 |
|---|---|
| `hello-nest.md` | **はじめての方はここから。** AIパートナーとの最初の挨拶（自己完結型・どのAIでも可） |
| `first-app.md` | 最初のアプリを「はじめてモード」（書類仕事なし）で一緒に作る |
| `resume-work.md` | 前回の続きから開発を再開させる（2回目以降の方用） |
| `implement-feature.md` | 仕様書をもとに機能を実装させる |
| `fix-bug.md` | バグを修正させる |
| `review-code.md` | コードレビューをさせる（実装したAIと別のAI推奨） |

## 運用ルール

- プロンプトは特定AIの独自構文（ツール固有コマンド等）を使わず、プレーンな指示文で書く。
- 繰り返し使う指示が生まれたら、会話で使い捨てにせずここにテンプレート化する。
