# 💡 knowledge/ — みんなの気づきメモ

> 🧭 ここは**転んだ記録と、立ち上がり方**の置き場です。誰かが一度困って解決したことは、ここに書き残されて、次の人の5分を救います。

## 使い方（引く人）

1. このフォルダのファイル名を眺める（`日付-内容.md` 形式）
2. 近そうなものを開く。**冒頭3行（状況→結論→理由）だけ読めば判断できる**ようになっています
3. 見つからなければ、🤖 AIパートナーにその画面を見せて聞く。解決したら——次はあなたが書く番です

## 書き方（残す人）

- 1つの気づき＝1ファイル。ファイル名は `YYYYMMDD-短い英語スラッグ.md`
- **冒頭に3行サマリを必ず書く**: 状況（何をしようとした）→ 結論（どうすればいい）→ 理由（なぜそうなる）
- そのあとに詳細手順・スクリーンショットの説明・参考リンクを自由に
- 実際に起きたことだけを書く（一般論の転載はしない。それはHandbookと検索の仕事）

## ここに書くもの／書かないもの

| 書く | 書かない（置き場所） |
|---|---|
| ハマった問題と解決法 | 恒久ルール → `docs/`（Handbook） |
| 調べてわかった意外な事実 | 設計判断の理由 → `docs/adr/` |
| 次もきっと使う小ワザ | 進行中の作業メモ → タスクファイルの作業ログ |
