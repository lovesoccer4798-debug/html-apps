# 「PRを作成できませんでした」— PRは比較先のブランチが無いと作れない

- **状況**: AIツールの「Create PR」ボタンを押したら「The requested GitHub resource was not found／プルリクエストを作成できませんでした」
- **結論**: リポジトリにブランチが1本しか無いのが原因。PR（プルリクエスト＝変更の取り込み提案）は「あるブランチ → 別のブランチ」の形でしか作れないので、先に `main` を作る
- **理由**: 最初のpushでできた1本目のブランチがデフォルトブランチになっており、比較する相手が存在しなかった

## 詳細

1. いまのブランチの内容を main として作る: `git push origin HEAD:refs/heads/main`
2. デフォルトブランチを main に変更する（`20260707-change-default-branch.md` 参照）
3. あらためてPRを作成する（作業ブランチ → main）

2026-07-07。Workspace v1.0の初回PR作成時に発生。
